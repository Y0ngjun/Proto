# 프로젝트 설계 (Design)

## 개요

Proto는 **쉽고 빠른 게임 프로토타입 개발**을 위한 경량 3D 게임 엔진입니다.

## 타겟 사용자

- 게임 핵심 플레이 검증을 원하는 기획자
- 게임 개발 입문자
- 빠른 프로토타입 개발이 필요한 개발자

## 시스템 요구사항

### 최소 사양
- **OS**: Windows 10 이상
- **메모리**: 4GB RAM
- **그래픽**: OpenGL 3.3 지원

### 개발 환경
- Visual Studio 2022 이상
- C++17 이상

## 기술 스택

- **언어**: C++17
- **그래픽**: OpenGL 3.3
- **윈도우**: GLFW 3.4
- **수학**: GLM 1.0.3
- **UI**: Dear ImGui (Docking)
- **데이터**: yaml-cpp

## 핵심 특징

- ✅ Object-Component 패턴 (확장성)
- ✅ 비주얼 에디터 (마우스 배치)
- ✅ 실시간 플레이 테스트 (Play 버튼)
- ✅ 씬 저장/로드 (YAML 포맷)

## 주요 기능

### 에디터 환경
- 3D 오브젝트 시각적 배치
- 씬 저장/로드
- 실시간 미리보기

### 게임 시스템
- GameObject-Component 기반 오브젝트
- 컴포넌트별 속성 실시간 편집
- Transform, Renderer, Camera, Light 등 기본 컴포넌트

### 물리 시스템 (향후)
- 충돌 감지
- 중력, 속도

### 스크립트 시스템 (향후)
- C++ 기반 ScriptComponent
- 게임 로직 작성

## MVP

**3D 탄막 게임 프로토타입** - 에디터로 만든 완성 게임
