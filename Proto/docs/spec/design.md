# design.md

프로젝트 기획

## 1. 프로젝트 개요

Proto는 쉽고 빠른 게임 프로토타입 개발을 위한 3D 게임 엔진입니다.  
쉽고 빠른 개발을 위해 노드 기반 프로그래밍과 에디터 환경을 제공합니다.  
경량 게임 엔진을 목표로 하기 때문에 프로토타입 개발에 필수적인 기능만 포함합니다.

## 2. 타겟 사용자

- 게임의 핵심 플레이 검증을 원하는 기획자
- 게임 개발을 처음 접하는 입문 개발자
- 그 외 모든 빠른 프로토타입 개발을 원하는 개발자

## 3. 시스템 요구사항

### 최소 사양
- **OS**: Windows 10 이상
- **프로세서**: Intel Core i5 이상
- **메모리**: 4GB RAM
- **그래픽**: OpenGL 3.3 지원 그래픽카드
- **스토리지**: 500MB

### 개발 환경 (Proto 개발용)
- Visual Studio 2022 이상
- C++17 이상

## 4. 기술 스택

- **언어**: C++17
- **그래픽**: OpenGL 3.3
- **윈도우 관리**: GLFW 3.4
- **수학**: GLM 1.0.3
- **UI**: Dear ImGui (Docking Branch)
- **빌드**: CMake (선택사항)

## 5. 프로젝트 특징

- 오브젝트-컴포넌트 패턴
- 에디터 인터페이스 제공
- 간단한 API 제공
- 노드 기반 프로그래밍 제공
- 불필요한 기능 제거(복잡한 렌더링 또는 애니메이션)
- 게임에 필수적인 기능만 포함

## 6. MVP

완성된 게임엔진으로 3D 탄막 게임 프로토타입 개발

## 7. 주요 기능

### 7.1 에디터 환경
- **비주얼 에디터**: 마우스로 3D 오브젝트 배치
- **씬 관리**: 게임 씬 저장/로드
- **실시간 미리보기**: 플레이 버튼으로 즉시 게임 테스트

### 7.2 게임 오브젝트 시스템
- **오브젝트 생성**: 씬에 오브젝트 추가
- **오브젝트 제거**: 게임 진행 중 오브젝트 삭제
- **오브젝트 속성 편집**: 인스펙터 패널에서 실시간 수정

### 7.3 컴포넌트 기반 구조
- **트랜스폼**: 위치, 회전, 크기 조절
- **메시 렌더러**: 3D 모양 표시
- **스크립트 컴포넌트**: 게임 로직 작성
- **머터리얼**: 색상, 텍스처 설정 (기초)

### 7.4 물리 시스템
- **충돌 감지**: 오브젝트 간 충돌 판정
- **물리 상호작용**: 중력, 속도, 가속도

### 7.5 스크립트 시스템
- **C++ 스크립트**: 고급 사용자를 위한 직접 코딩
- **노드 기반 프로그래밍**: 코드 없이 블록으로 게임 로직 (추후 개발 예정)

### 7.6 기본 렌더링
- **3D 렌더링**: OpenGL 기반 3D 그래픽
- **기본 도형**: 큐브, 평면, 구 등 제공

## 8. 사용 사례

- **3D 탄막 게임**: 프로토타입 제작
- **캐주얼 3D 게임**: 간단한 게임 프로토타입
- **게임 기획 검증**: 핵심 플레이 빠르게 테스트
- **게임 개발 학습**: 엔진 구조 이해하며 배우기
